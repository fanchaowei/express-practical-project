## 个人实战项目

本项目作为个人实战项目的后端

使用技术栈：
1. express + Prisma
2. 三层架构 + model
3. 数据库使用 postgreSQL + docker-compose

我打算做一个类似后台的系统，然后内部后续可以添加很多子系统模块，每个子系统都是导航栏中的一个菜单（易扩展）。
1. 系统需要有基础的登录流程（鉴权授权）
2. 目前系统内打算先做一个子系统：影片收藏子系统
   1. 该子系统允许我添加我喜欢的电影/电视剧/番剧/动画电影
      1. 可以传入电影/电视剧/番剧/动画电影的图片（可传入多张，其中一张为封面）、评分、标签 tag、上映年份、评语等。
   2. 可以删除电影/电视剧/番剧
   3. 可以通过类型（电影/电视剧/番剧/动画电影）、评分、标签 tag、上映年份等进行筛选。
